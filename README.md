A clean, minimalist, and impactful **Motivational Meme** web page built with HTML and CSS. This project focuses on typography, layout centering, and classic aesthetic styling to deliver a powerful message.

## ## Preview

> **"Try it. Only those who dare to try can succeed."**

The project features a dark mode aesthetic with a white-bordered image and high-quality typography using Google Fonts.

---

## ## Features

* **Custom Typography:** Utilizes the `Libre Baskerville` font from Google Fonts for a classic, sophisticated look.
* **Modern Styling:** * Dark background with contrasting white text.
    * 5px solid white border around the central image.
    * CSS `text-transform` applied for consistent header styling.
* **Centered Layout:** Uses precise CSS margin and width calculations to ensure the content is perfectly centered on the screen.
* **Responsive Image:** The image is set to fill its container while maintaining its aspect ratio.

---

## ## Built With

* **HTML5:** For the structured content.
* **CSS3:** For the custom styling and layout.
* **Google Fonts:** Specifically the *Libre Baskerville* and *Caveat* families.

## ## How to Run

1.  Clone this repository:
    ```bash
    git clone [https://github.com/oznc10/css-motivation-meme-project.git](https://github.com/oznc10/css-motivation-meme-project.git)
    ```
2.  Navigate to the project folder:
    ```bash
    cd css-motivation-meme-project
    ```
3.  Open `index.html` in your favorite web browser.

## ## Requirements Checklist

This project was built following specific design constraints:
* [x] Main heading using **Libre Baskerville**.
* [x] White text on a black background.
* [x] Centered `div` container with 50% width and 25% margin-left.
* [x] Image with a 5px white border.
* [x] Headings styled using `text-transform`.
